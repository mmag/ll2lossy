# ll2lossy

macOS-приложение для пакетной конвертации lossless-аудио в MP3. Двухпанельный интерфейс: слева выбираете источник, справа — папку назначения.

## Поддерживаемые форматы

**Конвертация → MP3:** FLAC, WAV, AIFF, ALAC, APE, WavPack, WMA, DSD (DSF/DFF), M4A  
**Копирование без конвертации:** MP3

## Возможности

- Рекурсивный обход папок, ленивая загрузка дерева
- Параллельная конвертация (до 16 потоков)
- Режимы кодирования: VBR (V0–V9) и CBR (128–320 кбит/с)
- Сохранение тегов ID3 и структуры подпапок
- Поведение при совпадении имён: пропустить / перезаписать / добавить суффикс
- Общий прогресс-бар и детальный список задач
- Drag & drop: перетащить файлы в правой панели в другую папку — переместит их
- Выделение, снятие выделения и удаление файлов в правой панели

## Требования

- macOS 14 Sonoma или новее
- Xcode 15+ / Swift 5.9+
- ffmpeg (устанавливается через `setup.sh`)

## Установка

```bash
# 1. Установите ffmpeg через Homebrew (если ещё нет)
brew install ffmpeg

# 2. Скопируйте бинарник в Application Support
./setup.sh

# 3. Соберите и запустите
swift run
```

`setup.sh` копирует ffmpeg из Homebrew в `~/Library/Application Support/ll2lossy/ffmpeg`. Приложение использует именно этот путь — системный ffmpeg не трогается.

## Использование

1. **Источник (левая панель)** — откройте папку с lossless-файлами, отметьте нужные галочками. Можно выбирать отдельные файлы или целые папки.
2. **Назначение (правая панель)** — откройте папку, куда сохранять MP3.
3. Нажмите стрелку **→** по центру или `⌘↩` — начнётся конвертация.
4. Прогресс отображается в статус-баре и в панели задач (кнопка в тулбаре).

Альтернативно можно перетащить файлы из левой панели прямо на правую.

## Настройки

Открываются кнопкой шестерёнки в тулбаре:

| Параметр | Варианты |
|---|---|
| Режим кодирования | VBR (V0–V9), CBR (128/192/256/320 кбит/с) |
| Теги ID3 | Сохранять / не сохранять |
| При совпадении имён | Пропустить / Перезаписать / Добавить суффикс (`_1`, `_2`, …) |
| Параллельных задач | 1–16 |
